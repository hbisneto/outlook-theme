# Outlook Theme

Outlook theme brings Microsoft Outlook to Visual Studio Code.

![Outlook for Visual Studio Code](https://raw.githubusercontent.com/hbisneto/hbisneto.github.io/main/outlook-theme/banner.png)

## Available Themes

- Outlook Theme (Dark)
- Outlook Theme (Light)
